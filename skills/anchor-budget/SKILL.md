# anchor-budget — 예산 집행률 분석

## 역할

RISE(앵커) 사업 예산 데이터를 분석하여 집행률을 계산하고, 개조식 요약 테이블을 생성한다.  
references/budget-codes.md의 예산 과목 코드를 참조한다.

## 집행률 계산식

```
집행률(%) = 집행액 ÷ 배정액 × 100
잔액 = 배정액 - 집행액
```

## 입력 형식

다음 중 하나의 형식을 허용:
1. CSV 텍스트 (헤더: 과목명, 배정액, 집행액)
2. 자연어 서술 ("○○ 항목에 ○○원 배정, ○○원 집행")
3. 마크다운 테이블

## 출력 형식

### 요약 테이블
| 과목 | 배정액 (백만원) | 집행액 (백만원) | 집행률 | 잔액 (백만원) | 상태 |
|------|----------------|----------------|--------|--------------|------|

### 개조식 요약
```
○ 예산 집행 현황 (기준일: YYYY. MM. DD.)
  - 총 배정액: ○○억 원
  - 총 집행액: ○○억 원 (집행률 ○○%)
  - 잔 액: ○○억 원
○ 과목별 현황
  - [과목명]: 배정 ○○백만 원 / 집행 ○○백만 원 (○○%)
```

## 상태 판정 기준

| 집행률 | 상태 |
|--------|------|
| 90% 이상 | 양호 |
| 70~89% | 보통 |
| 50~69% | 주의 |
| 50% 미만 | 부진 |

## 주의사항

- 미공개 예산 데이터 입력 시 처리 거부
- 개인별 급여·인건비 상세 내역 처리 금지
- 결과는 반올림하여 소수점 첫째 자리까지 표시

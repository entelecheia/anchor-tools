# anchor-minutes — 회의록 정리

## 역할

비구조화된 회의 노트 또는 녹취 텍스트를 RISE(앵커) 표준 회의록 형식으로 변환한다.  
references/minutes-template.md 형식을 준수한다.

## 입력 형식

- 자연어 회의 노트 (순서 무관)
- Plaud AI 녹취 텍스트
- 간단한 메모 (참석자·날짜 포함 권장)

## 처리 절차

1. 일시·장소·참석자 정보 추출 (명시되지 않으면 `[확인 필요]` 표기)
2. 발언 내용을 안건별로 분류
3. 논의 내용을 개조식으로 요약
4. 결정사항과 후속과제를 분리하여 목록화
5. 후속과제에 담당자·기한 정보가 있으면 반드시 포함

## 출력 형식

references/minutes-template.md 참조.

## 주의사항

- 개인 발언 내용 중 민감 정보(급여, 인사, 개인 의견 등)는 `[내용 생략]` 처리
- 참석자 명단의 소속·직위는 원문에 있을 때만 포함
- 결정사항이 없는 안건은 "논의 진행 중" 명시
- 영어 발언은 한국어로 번역하되 원문 병기 가능

## 사용 예시

```
사용자: 아래 메모를 회의록으로 정리해줘
        "3/15 오후 2시 2층 회의실. 참석: 이영준, 문현주, 조인범
         - 2차년도 계획 논의, 예산 편성 일정 확인
         - 조인범: 트랙별 정원 확정 필요 → 4월 초까지
         - 결정: 4/5 운영위원회에서 최종 확정"
```

# RISE(앵커) 공식 용어집 (한영 대역)

## 사용 규칙

- 이 용어집에 등재된 용어는 번역 시 반드시 해당 영문/국문 표기 사용
- 등재되지 않은 용어는 문맥에 맞게 번역 후 병기 권장
- 최초 등장 시: `앵커 사업 (Anchor Project)` → 이후 `Anchor Project` 단독 사용 가능

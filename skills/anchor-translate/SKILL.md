# anchor-translate — 한영 번역 (RISE 용어 일관성)

## 역할

RISE(앵커) 사업 관련 문서를 한국어 ↔ 영어로 번역한다.  
references/glossary.md의 공식 용어집을 엄격히 준수하여 번역 일관성을 보장한다.

## 번역 원칙

1. **용어 우선**: glossary.md에 등재된 용어는 반드시 해당 번역어 사용
2. **Cheju Halla University**: 대학명은 항상 이 영문 표기 사용 (Jeju Halla 금지)
3. **공식 문체**: 학술·행정 문서에 적합한 격식체 유지
4. **수치 보존**: 날짜, 금액, 인원 수 등 수치는 변환 없이 유지
5. **약어 처리**: 최초 등장 시 전체 명칭 + 약어 병기, 이후 약어 단독 사용 가능

## 번역 방향 감지

- 한글이 주를 이루면 → 영어 번역
- 영어가 주를 이루면 → 한국어 번역
- 혼용 시 사용자에게 방향 확인

## 출력 형식

기본적으로 번역문만 출력.  
용어 주석이 필요한 경우 각주 또는 괄호 병기 사용:
- 예: `앵커 사업 (Anchor Project, formerly RISE)`

## 지원 문서 유형

- 보고서, 공문, 보도자료
- 발표 자료 (슬라이드 텍스트)
- MOU, 협약서 (법적 효력 없음 — 공식 계약은 법무 검토 필수)
- 이메일, 안내문

## 주의사항

- 개인정보가 포함된 문서 처리 금지
- 번역 결과의 법적 효력 보장 불가 — 공식 문서는 반드시 전문 번역가 검토 권고
- 전문 기술 용어(항공우주, 의료 등) 불확실 시 원문 병기 후 확인 요청

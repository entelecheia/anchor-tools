# anchor-report — 개조식 보고서 변환

## 역할

일반 서술문 또는 회의 내용을 **한국 공공기관 표준 개조식** 형식으로 변환한다.  
references/report-guide.md의 규칙을 엄격히 준수한다.

## 작동 방식

1. 입력 텍스트를 문단/항목 단위로 분석
2. 각 문장을 명사형 종결어미로 변환 (예: "~했습니다" → "~함", "~이다" → "~임")
3. 불필요한 수식어, 접속사, 주어 제거
4. 계층 구조가 필요한 경우 `○ > - > ·` 불릿 체계 적용
5. 변환 결과를 마크다운 코드블록으로 출력

## 출력 형식

```
○ [대항목]
  - [중항목]
    · [세항목]
```

## 주의사항

- 수치, 날짜, 고유명사는 원문 그대로 유지
- "Cheju Halla University" 표기 준수 (Jeju Halla 사용 금지)
- 금액 단위: 백만 원 또는 억 원 (맥락에 따라)
- 보안 정보(개인정보, 급여, 미공개 예산) 포함 시 처리 거부 후 안내

## 사용 예시

```
사용자: 아래 내용을 개조식으로 변환해줘
        "지난 3월에 진행된 사업 설명회에는 약 150명이 참석하였으며,
         참가자들의 만족도가 높았습니다."

결과:
○ 3월 사업 설명회 개최 결과
  - 참석 인원: 150명
  - 참가자 만족도: 높음
```

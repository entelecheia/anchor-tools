# 개조식 보고서 작성 규칙

## 개요

개조식(個條式)은 한국 공공기관 문서의 표준 서술 방식으로, 각 항목을 독립된 명사구 또는 명사형 종결어미로 끝내는 방식이다.

---

## 핵심 규칙

### 1. 명사형 종결

| 원문 | 개조식 변환 |
|------|------------|
| ~했습니다 | ~함 |
| ~하였음 | ~함 |
| ~이다 / ~입니다 | ~임 |
| ~할 예정입니다 | ~할 예정 |
| ~되었습니다 | ~됨 |
| ~할 것입니다 | ~할 것임 |
| ~검토 중입니다 | ~검토 중 |
| ~추진하고 있습니다 | ~추진 중 |

### 2. 주어 생략

- 문맥상 명확한 주어는 생략
- 잘못: "사업단이 3월에 설명회를 개최함"
- 올바름: "3월 설명회 개최"

### 3. 불필요한 수식어 제거

- 잘못: "매우 다양한 분야의 여러 전문가들이 참여하여"
- 올바름: "각 분야 전문가 참여"

### 4. 불릿 계층 구조

```
○ 대항목 (사업·안건 단위)
  - 중항목 (세부 내용)
    · 소항목 (부연 설명, 수치)
```

### 5. 수치 표기

- 금액: `○○억 원` 또는 `○○백만 원` (단위 띄어쓰기)
- 날짜: `2026. 3. 15.(일)` 형식
- 비율: `○○%` (% 앞 띄어쓰기 없음)
- 인원: `○○명`

### 6. 공통 구조 패턴

#### 사업 추진 현황 보고
```
○ 추진 개요
  - 사업명: ○○○
  - 기간: ○○○○. ○○. ○○. ~ ○○○○. ○○. ○○.
  - 예산: ○○억 원
○ 주요 추진 실적
  - [항목1]
  - [항목2]
○ 향후 계획
  - [항목1]
```

#### 예산 집행 현황
```
○ 배정액: ○○억 원
○ 집행액: ○○억 원 (집행률 ○○%)
○ 잔액: ○○억 원
  - 사유: [사유 기재]
```

---

## 금지 표현

- "~에 대하여", "~와 관련하여" → 삭제하거나 명사형으로 대체
- "~하는 것이 필요합니다" → "~필요"
- "~라고 할 수 있습니다" → 삭제
- 중복 표현: "계획을 계획함" → "계획 수립"

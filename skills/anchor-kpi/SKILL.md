# anchor-kpi — KPI 성과지표 관리

## 역할

RISE(앵커) 사업의 정량·정성 성과지표를 입력받아 현황을 정리하고 HTML 대시보드 또는 개조식 보고서를 생성한다.  
references/kpi-framework.md의 지표 체계를 기준으로 한다.

## 입력 형식

- 지표명, 목표치, 실적치를 포함한 텍스트
- CSV 형식
- 자연어 ("참여 학생 수가 450명을 달성했어")

## 출력 옵션

사용자가 선택:
1. **개조식 요약** (기본): 마크다운 테이블 + 개조식 해설
2. **HTML 대시보드**: 진행률 바, 색상 코딩 포함 단일 HTML 파일

## 달성률 계산

```
달성률(%) = 실적치 ÷ 목표치 × 100
```

## 상태 색상 코딩 (HTML 출력 시)

| 달성률 | 색상 | 상태 레이블 |
|--------|------|------------|
| 100% 이상 | 초록 (#22c55e) | 목표 달성 |
| 80~99% | 파랑 (#3b82f6) | 양호 |
| 60~79% | 노랑 (#f59e0b) | 보통 |
| 60% 미만 | 빨강 (#ef4444) | 주의 |

## 개조식 요약 형식

```
○ KPI 달성 현황 (기준: YYYY. MM. DD.)
  - 총 지표 수: ○개 (정량 ○개, 정성 ○개)
  - 목표 달성: ○개 / 양호: ○개 / 주의: ○개
○ 주요 지표별 현황
  - [지표명]: 목표 ○○ / 실적 ○○ (달성률 ○○%)
```

## 주의사항

- 지표 정의가 불명확한 경우 사용자에게 확인 요청
- 정성 지표는 달성 여부(Y/N) 또는 단계(미착수/진행중/완료)로 표현
- 개인 단위 성과는 집계 수치로만 처리

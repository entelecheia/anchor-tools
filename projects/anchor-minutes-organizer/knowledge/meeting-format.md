# RISE(앵커) 회의 유형 및 형식 가이드

## 정기 회의 체계

### 운영위원회 (분기 1회)
- 구성: 위원장(총장), 부단장, 각 본부장, 외부 위원
- 주요 안건: 사업 추진 현황, 예산 집행 점검, 주요 의사결정
- 형식: 공식 의사록, 서명 필요
- 정족수: 재적 위원 과반수 참석, 참석 위원 과반수 의결

### 실무협의회 (월 1회)
- 구성: 부단장, 각 본부 실무 담당자
- 주요 안건: 월간 현황 공유, 현안 협의, 일정 조율
- 형식: 표준 회의록

### 본부별 팀 미팅 (주 1회 또는 격주)
- 구성: 본부장 + 본부 구성원
- 주요 안건: 주간 업무 보고, 진행 상황 점검
- 형식: 간소화된 회의록 (Action Items 중심)

### 외부 협력 회의
- 구성: 사업단 + 협력 기업·기관 담당자
- 주요 안건: 협력 현황, 신규 협력 논의
- 형식: 표준 회의록 (외부 참석자 정보 필수)

---

## 회의록 파일명 규칙

```
YYMMDD-[회의유형]-[핵심주제].md
```

예시:
- `260405-운영위원회-2차년도계획확정.md`
- `260315-실무협의회-예산집행점검.md`
- `260320-Pix4D협력-MOU체결협의.md`

저장 경로: `meetings/YYYY/YYYY-MM/`

```

```

### 외부 참석자
```
[성명] ([소속 기관] [직위])
홍길동 (Pix4D Korea 대표이사)
```

## 안건 번호 체계

```
안건 제1호: [제목] (보고/심의/의결)
안건 제2호: [제목]
```

- 보고: 정보 공유 목적
- 심의: 논의 후 결론 도출
- 의결: 공식 의사결정 (운영위원회만 해당)

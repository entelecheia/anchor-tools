# RISE 사업 예산 운용 규정 요약

## 1. 예산 편성 원칙

- 목적별 편성: 사업 목적에 직접 기여하는 항목만 편성
- 과목 간 전용 제한: 과목 간 전용은 사업단장 승인 필요 (10% 이상 변경 시 교육부 승인)
- 간접비 한도: 직접비의 20% 이내

## 2. 집행 기준

### 인건비 (100)
- 월별 정기 집행
- 연구원: 채용 공고 → 선발 → 계약 → 급여 지급 순서 준수
- 인건비 과집행 불가 (계약 금액 초과 지급 금지)

### 연구활동비 (300)
- 지출 품의서 작성 → 결재 → 집행 → 증빙 보관
- 30만 원 이상: 반드시 세금계산서 또는 카드 영수증
- 5만 원 이상 현금 지출: 간이영수증 불가, 세금계산서 필수

### 연구장비 (500)
- 500만 원 이상: 자산 등록 의무
- 구매 전 견적서 3개 이상 징구 (1,000만 원 미만) / 입찰 (1,000만 원 이상)
- 정부 조달 우선 원칙

### 국제협력비 (700)
- 해외 출장: 출장 계획서 → 사전 승인 → 귀국 후 결과 보고서
- 항공료: 이코노미 원칙 (장거리 비즈니스 예외 인정)
- 일비·숙박비: 공무원 여비규정 준용

## 3. 집행 금지 항목

- 개인 소비성 지출 (식사비 단독, 주류, 선물 등)
- 연구 목적과 무관한 물품 구매
- 현금 지급 후 영수증 미수취
- 사업 종료 후 집행 (정산 기간 내 처리 원칙)

## 4. 불용액 처리

- 연말 불용액 발생 시 불용 사유서 제출
- 불용액 5% 이내: 자체 처리
- 5~10%: 본부장 보고
- 10% 초과: 사업단장 보고 + 교육부 통보 + 차년도 배정액 조정 가능

## 5. 정산 및 감사 대비

- 모든 지출 증빙은 사업 종료 후 5년 보관
- 분기별 자체 감사 실시
- 교육부 현장 점검 대비: 집행 증빙 파일 체계적 관리 필수
- 연구비 카드 사용 원칙 (현금 최소화)

## 6. 주요 집행 일정

| 시점 | 행동 |
|------|------|
| 1월 | 연간 집행 계획 수립 |
| 3월 말 | 1분기 집행 점검 (목표: 20%) |
| 6월 말 | 2분기 집행 점검 (목표: 50%) |
| 9월 말 | 3분기 집행 점검 (목표: 70%) |
| 11월 | 연말 집행 촉진 강화 |
| 12월 15일 | 집행 마감 (잔여 처리) |
| 12월 31일 | 회계연도 종료 |

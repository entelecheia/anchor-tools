# 행정 업무 워크플로

## 출장보고서 작성 워크플로

### 입력 정보

담당자가 제공해야 할 최소 정보:

```
- 출장자: [이름/직급]
- 출장지: [장소]
- 기간: [YYYY.MM.DD ~ YYYY.MM.DD]
- 목적: [출장 목적]
- 동행자: [있으면 명시]
- 주요 활동: [일자별 활동]
- 결과 및 후속 조치: [핵심 내용]
```

### 처리 단계

**Step 1 — 개요 구성**
- 출장 기본 정보를 표준 헤더 형식으로 정리
- 기간 계산: 박수·일수 자동 산정

**Step 2 — 활동 정리**
- 일자별 활동을 개조식으로 변환
- 불필요한 서술 제거, 핵심 행동 중심 기술

**Step 3 — 성과 도출**
- 협의·결정 사항 명사형 종결로 정리
- 후속 조치 사항 별도 항목으로 분리

**Step 4 — 검토 요청**
- 생성된 초안에 "담당자 검토 후 발송" 주석 포함
- HWP 변환 필요 시 anchor-hwp 스킬 안내

## 공문 작성 워크플로

### 공문 유형별 처리

| 유형 | 특이사항 |
|------|----------|
| 협조 요청 | 요청 사항·기한·담당자 명시 필수 |
| 결과 통보 | 근거 문서 번호 포함 |
| 제출 안내 | 붙임 파일 목록 포함 |
| 알림 공문 | 일정·장소·대상 명확히 |

### 공문 번호 처리

공문 번호는 담당자가 직접 부여 (AI 임의 생성 금지):
```
[기관 기호]-[연도]-[일련번호]
예: 앵커사업단-2026-042
```

## 예산 집행 분석 워크플로

### 집행률 계산

```
집행률(%) = 집행액 ÷ 배정액 × 100
```

### 상태 기준

| 집행률 | 상태 | 조치 |
|--------|------|------|
| 90% 이상 | 정상 | — |
| 75~89% | 주의 | 집행 촉진 필요 |
| 75% 미만 | 경고 | 사업단장 보고 + 집행 계획 수립 |

### 연말 목표

- 집행률 목표: **90% 이상**
- 불용액 10% 초과 시: 사업단장 보고 + 교육부 통보 필요

### 입력 형식

CSV 또는 표 형식:
```
항목, 배정액(원), 집행액(원)
인건비, 500000000, 430000000
운영비, 200000000, 195000000
```

## 주간보고 취합 워크플로

### 수집 채널

- 메신저 `#anchor-weekly-report` 채널
- 이메일 수신 (담당자 직접 수집)
- 각 본부 담당자 직접 입력

### 취합 형식

**제출 마감**: 매주 금요일 오후 5시

**본부별 표준 항목**:
```
이번 주 주요 업무:
- [완료 업무 1]
- [완료 업무 2]

다음 주 예정:
- [예정 업무 1]

특이사항/요청:
- [있으면 명시]
```

### 종합 처리

1. 각 본부 보고를 표준 형식으로 통일
2. 사업단 전체 주요 일정 별도 항목으로 정리
3. 특이사항·요청 사항 운영지원본부 검토 후 처리

## 결재 문서 작성 워크플로

### 결재 문서 유형

| 유형 | 구성 |
|------|------|
| 시행 계획 | 목적·내용·일정·예산·기대효과 |
| 구매 요청 | 품목·수량·단가·합계·사유 |
| 출장 신청 | 출장자·기간·목적·경비 |
| 행사 결과 보고 | 개요·진행 결과·참가자·예산 집행 |

### 결재 라인 (참고)

```
담당자 → 팀장 → 본부장 → 사업단장 → (부단장) → 총장
```

금액·중요도에 따라 결재 라인 조정 — 담당자 확인 필수.

## MS Office 365 연동

### Word 공문 작성

- 에이전트 생성 MD → Word로 변환 후 공문 양식 적용
- 교육부 공문 양식은 `templates/` 폴더 참조

### Excel 예산 현황

- 에이전트 분석 결과 → Excel 복사·붙여넣기 또는 CSV 저장
- 피벗 테이블 활용 권장 (월별·항목별 집계)

### Teams/Outlook 연동 (선택)

- MCP Outlook 서버 설정 시 이메일 기반 주간보고 자동 수집 가능
- 설정 방법: `references/mcp-setup.md` 참조 (추후 추가)

## 분석 결과 활용

| 출력물 | 활용처 |
|--------|--------|
| 출장보고서 초안 | 결재 후 공문 발송, 성과 기록 |
| 공문 초안 | 담당자 수정 → 결재 → 발송 |
| 예산 현황 | 월간 예산 회의, 교육부 보고 |
| 주간보고 종합 | 부단장 주간 브리핑, 운영위원회 |

### 보관 경로

```
admin/reports/YYYY/        — 보고서
admin/official-docs/YYYY/  — 공문
admin/budget/YYYY/         — 예산 현황
admin/weekly-reports/YYYY/ — 주간보고
```
